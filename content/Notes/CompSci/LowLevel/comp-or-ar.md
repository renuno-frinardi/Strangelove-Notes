## I. Brief Introduction to Computer Systems

Computer systems consist of hardware and software that cooperate to run different program application. In a computer all the information is represented as a group of **bits** called **byte** (sequence of 8 bits, or a sequence of 8 1 or 0). Programs are translated into another form by other programs, from an ASCII text translated by compilers and linkers into binary executable files

Processors read and run binary instructions that are stored in main memory (RAM). Because computer works by moving information from I/O devices to main memmory (RAM) to registers, there exist a heirarchy that is ordering all the storage based on access time and capacity of each storage

The operating system kernel of a computer serves as an intermediary between application and hardware. It provides 3 fundamental abstraction:
- Files are abstraction for I/0 devices
- Virtual memory are abstraction from main memory (RAM) and disks
- Processes are abstraction of the processor, main memory, and I/O devices
